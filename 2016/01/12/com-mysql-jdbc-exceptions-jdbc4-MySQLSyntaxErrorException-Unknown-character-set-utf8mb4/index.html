使用成c3p0,或者出现一系列的连接池时出现
com.mysql.jdbc.exceptions.jdbc4.MySQLSyntaxErrorException: Unknown character set: ‘utf8mb4’

第一时间看不懂，百度，发现全是说数据库不支持这个字符集，打开MySQL配置文件一看，utf-8没错啊到底什么原因，伴随着这个异常的还有种情况就是链接很慢，我当就想了下，链接很慢会不会是jar的问题，看了下jar包版本的5.1.20,感觉太低了，试了试换了一5.1.6的版本，我擦奇迹的事情发生了居然正常了，好吧其实就是jar版本过低导致的不兼容。。
